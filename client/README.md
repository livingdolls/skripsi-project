## Deployment

To deploy this project run

```bash
  yarn run deploy
```

## Run Locally

Clone the project

```bash
  git clone https://github.com/livingdolls/frontend-information-retrieval.git
```

Go to the project directory

```bash
  cd frontend-information-retrieval
```

Install dependencies

```bash
 yarn  install
```

Start the server

```bash
  yarn start
```
